### DESCRIPTION
일정 시간마다 자동으로 position 변경이 일어나는 ViewPager입니다.

1. 상속받은 클래스, 인터페이스
- FrameLayout
- Runnable
- View.OnTouchListener
2. 기능
- Page를 좌우로 일정시간마다 움직임
- View가 touch된 경우 일시정지
3. 제한사항
- View.addView()로 사용 가능
- xml 직접 사용 불가능

- 프레임워크: Android
